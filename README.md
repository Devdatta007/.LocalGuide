# Satara Local Guide

A fully responsive web application showcasing the beauty, culture, and attractions of Satara, Maharashtra.

## Features

- **Fully Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, intuitive interface with smooth animations
- **Interactive Navigation** - Mobile-friendly hamburger menu
- **Sections Include:**
  - Hero section with call-to-action
  - Popular attractions (Ajinkyatara Fort, Thoseghar Waterfall, Kaas Plateau, Sajjangad Fort)
  - Local cuisine highlights
  - Culture and heritage information
  - Contact form
  - Social media links

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Font Awesome Icons

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required
3. Works offline once loaded

## Customization

- **Colors**: Edit CSS variables in `styles.css` (`:root` section)
- **Content**: Update text and information in `index.html`
- **Images**: Add background images to hero section or cards as needed
- **Functionality**: Extend features in `script.js`

## Browser Support
