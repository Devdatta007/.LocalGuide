# Restaurant Images

Add images with these exact filenames:
- `hotel-radhika.jpg` - Hotel Radhika restaurant
- `hotel-palvi.jpg` - Hotel Palvi Satara
- `manas-resto.jpg` - Manas Resto (iconic Satara brand)

## Image Guidelines
- **Format**: JPG, PNG, or WebP
- **Recommended size**: 800x600px or similar aspect ratio (4:3)
- **File size**: Keep under 500KB for optimal loading
- **Quality**: High quality but web-optimized

These images will be displayed with the same card layout and hover effects as the attractions section.
