# Logo Image

Add your team logo with this exact filename:
- `team-logo.png` - Your team logo (recommended)

## Image Guidelines
- **Format**: PNG (with transparent background recommended) or JPG
- **Recommended size**: 150x50px or similar (width x height)
- **Aspect ratio**: Horizontal/wide format works best
- **File size**: Keep under 100KB for fast loading
- **Background**: Transparent PNG works best for professional look

## Where it appears:
- Navigation bar (top left, next to "Satara Guide" text)
- Automatically resizes to fit the navbar
- Visible on all pages

## Alternative names accepted:
If you name your file differently, update the path in index.html:
- team-logo.png
- logo.png
- team-logo.jpg
