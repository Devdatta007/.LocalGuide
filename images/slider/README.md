# Slider Images

Add 4 images for the slider with these exact filenames:
- `slide1.jpg` - Beautiful Satara view
- `slide2.jpg` - Historic landmarks
- `slide3.jpg` - Natural beauty
- `slide4.jpg` - Cultural heritage

## Image Guidelines
- **Format**: JPG, PNG, or WebP
- **Recommended size**: 1920x500px (landscape orientation)
- **Aspect ratio**: 16:9 or similar wide format
- **File size**: Keep under 500KB for optimal loading

## Slider Features
- Auto-plays every 4 seconds
- Click left/right arrows to navigate
- Click dots at bottom to jump to specific slide
- Hover to pause auto-play
- Keyboard navigation (arrow keys)
- Smooth fade transitions
- Fully responsive on all devices
